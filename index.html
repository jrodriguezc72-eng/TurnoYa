Seguridad y privacidad

Funciones de protección para TurnoYa

Diseñamos la plataforma para que la información personal se comparta solo cuando hay un servicio confirmado y la confianza está establecida.

Perfiles con datos protegidos

El nombre completo, DNI y teléfono solo se muestran cuando ambas partes aceptan la asignación.

Verificación de identidad

Validación de técnicos, clínicas y familias antes de permitir el uso de la plataforma.

Chat interno seguro

La comunicación inicial se realiza dentro de la aplicación sin revelar datos personales.

Control de acceso por roles

Cada usuario solo ve la información necesaria para su función.

Reporte de usuarios sospechosos

Denuncia perfiles, mensajes o solicitudes que parezcan fraudulentos de inmediato.

Calificaciones y reseñas

Solo los usuarios que completaron un servicio pueden dejar opiniones.

Autenticación de dos factores

Una capa extra de seguridad al iniciar sesión en usuarios clave.

Alertas de actividad inusual

Notificaciones cuando se detectan accesos desde ubicaciones o dispositivos no habituales.

Cifrado de datos

La información personal y las comunicaciones se almacenan y transmiten cifradas.
